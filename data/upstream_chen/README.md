# Chen Peng efficiency-curve snapshot

`chen_efficiency_curves_origin_sheet5.csv` is a row-oriented snapshot of the
three curves in `Book7/Sheet5` of Chen Peng's Origin project `第四章.opju`.
It contains 23 current-density samples for each 270/300/330-cell stack.

The power column is the gross electrochemical stack power returned by
`get_condition.m`.  The efficiency column is the LHV system efficiency returned
by `cal_eff.m`; its numerator subtracts compressor power.  These two columns do
not share the same power basis, so downstream allocation must use the audited
net-power reconstruction generated by `scripts/74_build_chen_efficiency_curve_audit.py`.

The sampled domains are modeling-curve domains.  They are not independently
validated physical minimum or maximum operating limits.
