# FC-only退化机制消融

本实验在冻结动作路径上分别缩放连续、启停和变载退化。网格为3个连续项倍率与
5x5个启停/变载倍率，共75点。它用于诊断结论是否由系数驱动，不假设系数变化后
原控制动作仍为闭环最优。

|:-----------------------------|----------------------------:|--------------------------:|-------------------------------:|

| ('empirical_1s', 'rotating') |                  -46.2168   |                      -100 |                     418.416    |
| ('zuo_fast_30s', 'average')  |                   -1.16058  |                         0 |                      -0.337801 |
| ('zuo_fast_30s', 'rotating') |                  -49.2113   |                      -100 |                     436.865    |
| ('zuo_slow_30s', 'average')  |                   -0.707333 |                         0 |                      -0.775278 |
| ('zuo_slow_30s', 'rotating') |                  -50.0779   |                      -100 |                      14.8514   |

默认系数下，Instant相对Average的总退化均值只降低0.66%-1.16%，三个场景均为
4/5个种子获胜。跨完整网格，逐样本获胜率为60%-80%，因此短时延寿优势不稳健。

Instant相对Rotating通常降低约46%-50%总退化，但该差异主要来自固定轮换产生的
启动事件。当启停系数为0且Zuo-fast变载系数较高时，Rotating可以反超Instant。
这说明固定轮换不是可靠主方法，也说明当前Instant不能仅凭短时总退化成为论文贡献。
