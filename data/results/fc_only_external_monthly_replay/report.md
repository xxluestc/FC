# 独立跨月实车功率回放

- 2025-06至2026-06共13个月，每月前/中/后三个预声明时间层各取1个连续块，共39块、70,200个基础1秒功率点；3种健康身份和2种冻结策略形成234例、421,200个闭环评估步，其中156例构成78个有信息的配对比较。
- 这些块不与原七天链拼接，每块重置健康和控制状态；控制器、40 kW参考、健康参数和慢层规则均未重调，也没有使用未来需求。
- 原始遥测按与七天canonical相同的规则去重并在不超过10秒的段内重采样；70,200步中35,117步是相邻遥测包之间的插值，任何大于10秒的缺口都没有跨越。负功率噪声0步按燃料电池不可吸收回馈功率映射为0。
- 414个高于40 kW的点来自2个月，均没有截断，最高观测功率41.164 kW；最大跟踪误差5.496 kW，硬约束违规0步。
- 主统计单位仍是13个日历月：先在每个轨迹块内平均“最老堆为0/1”两个有信息的身份循环，再汇总同月三个时间层，避免把身份复制和同月块当成独立样本。
- Health-greedy相对固定双堆的终端最大退化变化均值为-0.003938848个百分点，月级BCa bootstrap 95%区间[-0.004063830, -0.003848424]；改善13/13个月，单侧精确Wilcoxon和符号检验均为p=0.00012207。该p值是13/13同号时的精确下界，效应大小以均值和区间为准。
- 对应总期望退化变化+0.000284514个百分点、跟踪MAE变化-0.0494 kW、氢耗强度变化-0.1147 g/kWh；安全驻留覆盖261步，占全部评估步0.062%。
- 该实验补强跨月真实功率可执行性和健康均衡方向，不证明同一物理电堆跨13个月连续老化，也不把30分钟块外推为寿命。
