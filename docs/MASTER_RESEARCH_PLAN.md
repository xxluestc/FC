## 1. 当前研究问题

当前首先解决三堆PEMFC系统内部的运行堆选择和功率分配：

```text
当前已知需求功率
+ 三个电堆各自的D/theta/IV状态、电流、开关和驻留状态
        ↓
候选动作[I1, I2, I3, on1, on2, on3]
        ↓
三堆N+1硬约束：最多两个堆在线，一个堆轮换休息
        ↓
实际执行动作驱动各堆确定性健康更新
D_i(k+1) → theta_i(k+1) → V_i/P_i(k+1)
        ↓
满足燃料电池侧目标功率，同时权衡氢耗、退化、性能损失、启停和变载
```

近期不把锂电池、SOC恢复和未来需求预测放入主决策问题。它们是后续外层优化，
不能阻塞燃料电池内层基础闭环。

## 2. 冻结边界

除非形成新的书面决策，以下口径保持不变：

| 项目 | 当前口径 |
|---|---|
| 系统结构 | 三堆适度超容量N+1，同一时刻最多两个堆在线 |
| 当前决策层 | 多个燃料电池之间的运行堆选择和功率分配 |
| 当前负载信息 | 只使用当前需求，不做未来需求预测 |
| 锂电池 | 暂不进入当前内层决策；保留未来外层接口 |
| 健康更新顺序 | 确定性动作更新 → Gamma随机退化 → 实车观测校正 |
| 单堆负载字段 | `liu_vehicle_canonical_1s.csv:fc_input_power_kw` |
| 标定/留出边界 | segment 0-21标定；segment 22-45完整留出 |
| 退化代理口径 | `C_deg(I|theta)`是当前健康下性能损失代理，不是材料退化直接观测 |
| Gamma口径 | 秒级控制用条件均值；完整分布只进入小时级风险/维护分析 |
| 文献系数 | Ghaderi/Pei系数必须标来源，不称为LZW数据因果辨识 |
| 高级算法 | 基础门槛通过前不训练Dreamer/EAWM/TD-MPC类模型 |

### G5：Gamma随机退化与统计评价

状态：基础诊断已完成；Gamma保留为聚合暴露不确定性，不作为120秒控制排序的采样指标。

- 完成三类负载、10个联合配对种子、三策略共90次在线随机健康运行，所有运行零硬违规；
- 负载种子与健康种子一一映射并在策略间共享，但不是独立双因素全排列；
- 120秒连续Gamma shape总量仅约`8.3e-4`-`1.5e-3`，90/90次运行的采样连续增量低于`1e-15%`；
- 解析时间尺度审计显示，120秒采样低于条件均值1%的概率约98.7%-99.0%，不适合估计均值；
- 冻结动作暴露聚合到1000小时后shape约26-37，可用于Gamma时间尺度诊断；
- 聚合实验中离散事件项占总期望退化约95%-99%，CV扫描没有改变策略方向，排序主要由事件系数主导；
- 旧1000小时实验同时重复了120秒内的启停事件，隐含不合理高频启停，已降级为历史诊断，不能支撑寿命或方法有效性；
- 控制决策继续使用确定性条件期望；Gamma用于长时间尺度敏感性，不声称寿命预测。

  最差堆改善方向不依赖30 kW截峰；全年独立审计随后支持40 kW作为经验运行参考，但不改变
  铭牌额定净功率未证明的边界；
- 逐段删一后两项主检验全部保留显著，最大Holm校正`p=0.015625`；16个单段符号反转压力情景
  有6个失去显著性，说明现有观测不由单段驱动但外推仍受`n=8`限制；
- 冻结最大误差案例的定向容差扫描中4.90 kW及以下失败、4.95 kW成功，5.5 kW相对当前最小
  测试成功值有0.55 kW余量；该结果解释工程容差但不声称全留出更紧容差均可行；
- 四种冻结开发暴露与三种健康身份的12个慢层决策点中，health-greedy、Expected-max和
  Gamma-CVaR在线集合12/12一致，两类目标最大遗憾均为0；复杂目标自身最优只作实现一致性，
  不作为独立性能贡献；
- 重新按Zuo 2024的N+1运行定义审计后，系统比较终点改为“第二个堆达到标定边界”；第一个堆
  越界后移出候选集，剩余两堆继续运行。该边界仍是LZW标定轨迹终点，不是物理失效阈值；
- 纯第二大损伤目标会主动牺牲一个堆。200种子开发结果中，它使首边界平均提前728.045小时，
  第二边界仅增加2.005小时且95%区间[-0.812,4.844]跨0，因此不进入主方法；
- 新增序统计量Blend 0.50，等权组合投影最大损伤和第二大损伤。200种子开发集相对固定双堆
  首边界增加74.640小时，第二边界增加4.790小时，95%区间[2.730,7.952]，39胜147平14负；
- 权重只在开发模板上筛选并冻结。13个月各3个外部真实块转为快层闭环暴露后，每月20种子验证
  得到首边界平均增加71.069小时，月级95%区间[46.254,100.305]；第二边界增加4.077小时，
  区间[1.604,7.438]，9月改善、1月持平、3月下降；月级符号检验`p=0.072998`，因此只声称
  平均效应为正，不声称绝大多数月份均改善；
- Blend 0.50成为当前慢层候选。它的主要价值是推迟首边界且不牺牲N+1第二边界，第二边界
  相对增益约0.3%，不是寿命跃升；高级算法只有在物理边界、异质性或维护目标更明确后再进入。
- 预声明41场景参数矩阵覆盖初始健康排列、Zuo单老化堆、文献GP随机效应分位数、边界±20%、
  Gamma CV 5%/20%、风险窗口和重调度周期。原始Blend在强异质性错配场景的第二边界平均
  `-47.20 h`且95%区间低于0，不能称普适鲁棒；
- Guarded Blend仅在初始损伤最大堆与估计退化速率最大堆身份一致且健康有区分时启用Blend，
  否则回退固定双堆。41场景中无第二边界负CI场景，最差均值`-0.50 h`且区间跨0；代价是
  跨场景第二边界平均收益由111.27 h降为17.92 h，属于保守适用域控制；
- 13个月异质性跨月验证中，参考场景首/第二边界为`+71.07/+4.08 h`；强随机效应对齐场景为
  `+50.77/+1.16 h`且第二边界区间跨0；强错配场景260个种子-月份组合全部精确回退固定基线；
- 两阶段RUL、第二边界连续性阈值和局部一步双边界安全投影均已定向否决。后者虽大幅推迟首边界，
  但在当前异质性排列和强GP随机效应场景仍出现第二边界负CI，不能替代Guarded Blend；
- 所有GP随机效应因子都是文献驱动压力分布的归一化分位数，不是21UBE0022车辆或三只真实堆的
  拟合参数。部署门控需要在线observer或维护检测给出各堆相对退化速率估计。
