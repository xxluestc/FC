## 关键判断

| 判断 | 当前结论 | 边界 |
|---|---|---|
| 主负载时间基准 | 实车1秒经验矩阵 | Zuo 30秒仅是压力场景工程假设 |
| 当前确定性健康基线 | Instant-health | Beam无稳定物理优势且约增加107%规划时间 |
| Average基线 | 保留对照 | 真实窗口中22/24成功，不能称普遍可行 |
| Rotating基线 | 保留公平性对照 | 轮休更均衡，但启停退化代价明显 |
| Gamma在线采样 | 不用于120秒策略均值排序 | shape约`1e-3`，100%运行连续采样近零 |
| Gamma不确定性 | 进入小时级风险/维护层 | 旧1000小时重复结果仅作历史诊断 |
| 当前方法证据 | 秒级Instant不足以单独成文 | 相对Average仅小幅且系数敏感 |
| 慢层主方法 | 24小时Guarded Blend候选 | 适用域内执行Blend 0.50，健康-速率错配时回退固定双堆；不使用未来负载 |
| 参数鲁棒性 | 原始Blend不能称普适 | 41场景中强GP随机效应错配使第二边界-47.20 h且CI低于0；Guarded Blend无负CI场景 |
| 异质性跨月验证 | 门控避免已知错配损失 | 参考场景第二边界+4.08 h；强对齐+1.16 h且CI跨0；强错配260/260种子精确回退0 |
| 两阶段RUL/局部安全投影 | 不进入主方法 | 两阶段策略增加切换并损害第二边界；双边界一步投影在两个场景仍出现负CI |
| 纯N+1第二边界目标 | 不进入主方法 | 200种子第二边界区间跨0，且首边界平均提前728.05 h；跨月首边界提前563.82 h |
| Gamma-CVaR | 降级为离线消融 | 四场景均未稳定超过确定性方法 |
| 滞回增强 | 不进入主方法 | 无阈值通过冻结的逐种子非劣规则 |
| 长期结果边界 | 仍是边界到达仿真 | 120个开发模板；LZW终点不是失效阈值 |
| 完整真实留出 | 冻结health-greedy方向有效且全例可执行 | 8段segment级统计成立；最长段7.543 h，只验证入口选择和段内快层；高于30 kW样本被截峰 |
| 负载容量口径 | 40 kW为全年实车经验参考，不是铭牌额定功率 | 目标40 kW出现291,209次；电堆侧p99=39.315 kW；最终链用40 kW无截峰，30 kW保留敏感性 |
| 实车健康信号 | 可监测，暂不可直接校正`D` | 两次时期台阶；最终时期原始-1.984%/100天，环境校正后-0.182%，95% CI含0 |
| 跨月外部功率验证 | 健康均衡方向稳定，但存在很小总退化代价 | 13/13月最大退化改善；每块重置健康，不能解释为13个月寿命试验 |
| 电池与未来预测 | 暂不进入当前优化 | 后续作为外层与独立优化点 |
