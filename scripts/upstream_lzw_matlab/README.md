# 刘占伟 IV/UKF 上游脚本

本目录保存当前退化代理和后续健康模型讨论中最常用的轻量上游脚本。

| 文件 | 来源/作用 |
|---|---|
| `iv_model.py` | 根据刘占伟 MATLAB IV 模型翻译的 Python 版本；用于计算 `V_model(I, θ)` |
| `UKF_PF.m` | 刘占伟 UKF-PF 老化参数辨识相关 MATLAB 脚本 |
| `UKF_test_all.m` | 刘占伟滤波算法目录中的 UKF 测试脚本 |

注意：

- 这些脚本用于代码逻辑审计和后续复现准备。
- 当前仓库没有上传完整 MAT 观测输入；因此不能仅凭本目录独立重跑全部 UKF-PF。
- 若后续目标变成“完全复现原始观测 → θ(k)”，需要继续补充 `data_mark_update`、`a/b/bestX`、电压观测 `V/v_test` 等 MAT 文件及其字段说明。
